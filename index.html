Октант-орнамент

Текст перетворюється на потік бітів (0 і 1) і порядково заповнює октант — трикутну восьму частину поля. Кожна одиниця дзеркалиться 8 разів навколо центра. Кількість рядків залежить від довжини тексту. Чорно-біле, без згладжування.

141×141
Символів: 0 · біт: 0 Рядків октанта: 0 / 0 Живих пікселів: 0 Бітів більше, ніж уміщає октант — збільш поле Октант — це ⅛ квадрата. Радіус = (поле−1)/2, тож рядків уміщається стільки ж.
